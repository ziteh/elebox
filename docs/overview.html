Overview

Overview

Elebox is an out-of-the-box electronic parts inventory management tool for those who want to manage their electronic parts efficiently, but find other management tools too complicated and cumbersome.

It uses an embedded database, eliminating the need to set up or installation database. The single-file database is easy to backup and supports exporting and importing data in YAML format, ensuring that your data is flexible and free.
