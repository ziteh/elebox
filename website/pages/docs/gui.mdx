# GUI

Graphical user interface edition of Elebox, powered by [Tauri](https://tauri.app/) + [Vue](https://vuejs.org/) + [Vuetify](https://vuetifyjs.com/).

![elebox screenshot](https://i.imgur.com/i6ihU9o.png)

## config.yaml

```yaml copy filename="config.yaml"
language: en
database: path\to\elebox.db
```

Path:

- Windows: `C:\Users\USERNAME\AppData\Local\elebox\config.yaml`
